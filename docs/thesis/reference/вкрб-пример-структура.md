## Как использовать

- Полный извлечённый текст: `docs/thesis/reference/вкрб-пример.txt`.
- Для анализа конкретного раздела сначала читать диапазон строк вокруг нужного заголовка.
- Если в разделе есть таблицы, диаграммы, рисунки, схемы, формулы или важное оформление, читать полный раздел из оригинального файла `docs/ВКР_пример.docx`, потому что текстовая копия нужна только для навигации и может терять визуальную структуру.
- Не загружать весь DOCX/PDF в контекст без необходимости.
