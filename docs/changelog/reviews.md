# Отзывы и рейтинги

## Функциональные изменения

- На карточках товаров и странице товара отображается средняя оценка и количество одобренных отзывов.
- На странице товара добавлен список одобренных отзывов покупателей.
- Авторизованный пользователь может отправить отзыв с оценкой от 1 до 5 и текстом.
- Новый отзыв получает статус `PENDING` и появляется на витрине только после модерации.
- Повторный отзыв одного пользователя на один товар запрещен.
- Неавторизованный пользователь при отправке отзыва перенаправляется на страницу входа с параметром возврата.
- В админ-панель добавлен раздел `/admin/reviews` для просмотра, фильтрации и модерации отзывов.
- Администратор может одобрить или отклонить отзыв.

## Технические изменения

- Использована существующая Prisma-модель `Review` и enum `ReviewStatus`.
- Добавлены `ReviewRepository` и `ReviewService` для работы с отзывами.
- Добавлены API routes:
  - `GET /api/products/[id]/reviews`;
  - `POST /api/products/[id]/reviews`;
  - `GET /api/admin/reviews`;
  - `PATCH /api/admin/reviews/[id]`.
- API routes используют серверную авторизацию и Zod-валидацию входных данных.
- Публичная выдача отзывов возвращает только отзывы со статусом `APPROVED` и только для активных товаров.
- Средний рейтинг и количество отзывов рассчитываются по одобренным отзывам.
- Изменение статуса отзыва записывается в журнал административных действий.
- Добавлены компоненты `RatingSummary` и `ProductReviews`.
