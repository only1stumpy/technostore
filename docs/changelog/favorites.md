# Избранное

## Функциональные изменения

- Авторизованный покупатель может добавлять товары в избранное из карточек товаров.
- Повторное добавление товара в избранное не создает дубликаты.
- Покупатель может удалять товар из избранного повторным нажатием на сердечко.
- Добавлена страница `/favorites` со списком сохраненных товаров.
- Неавторизованный пользователь при попытке добавить товар в избранное перенаправляется на страницу входа с параметром возврата.
- В шапке сайта добавлена иконка сердца для перехода в избранное.
- На странице товара добавлена кнопка добавления в избранное рядом с кнопкой корзины.
- В шапке сайта добавлен бейдж с количеством товаров в избранном для desktop и mobile меню.

## Технические изменения

- Добавлены repository и service для избранного: `FavoriteRepository` и `FavoriteService`.
- Добавлены API routes `GET /api/favorites`, `POST /api/favorites/items`, `DELETE /api/favorites/items/[productId]`.
- Запросы избранного возвращают только активные товары без soft-deleted товара, категории и бренда.
- Добавлен клиентский компонент `FavoriteButton` с optimistic UI и обработкой 401 redirect.
- Добавлен Zustand-store избранного для синхронизации состояния кнопок и счетчика в шапке.
- `ProductCard` перестроен так, чтобы кнопка избранного не была вложена внутрь ссылки на товар.
- Добавлена Prisma migration для новых моделей вовлечения, промокодов, отзывов, журнала действий и новых полей заказа.
